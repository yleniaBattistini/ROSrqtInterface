# ROSrqtInterface

## Launch rqt existing perspective 

#### Launch ROS process
```bash
roscore
```
#### Launch rqt existing perspective
```bash
rqt --perspective PERSPECTIVE_FILE
```


## Launch plugin
#### Point to the folder "catkin_ws/src/rqt_mypkg/src/rqt_mypkg 
```bash
rosrun rqt_mypkg rqt_mypkg
```
